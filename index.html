О проекте

Этот учебный сайт создан в рамках курса веб-разработки и представляет собой коллекцию практических заданий, демонстрирующих различные аспекты создания современных веб-интерфейсов. Проект включает в себя пять основных заданий, каждое из которых направлено на освоение конкретных технологий и техник вёрстки.

Первое задание "Пчела" демонстрирует возможности чистого CSS для создания графических элементов. С помощью позиционирования, градиентов, псевдоэлементов и clip-path была создана миниатюрная сцена с пчелой, облаками, грибами и травой. Это задание показывает, насколько выразительными могут быть стили без использования изображений или JavaScript.

Второе задание "Магазин игрушек" представляет интерактивную сцену с использованием CSS-анимаций и чекбоксов для создания интерактивности. При наведении на элементы запускаются анимации движения, а при клике на надпись появляется мальчик, который бежит через сцену. Это задание на работу с ключевыми кадрами и трансформациями.

Третье задание "Основы js" переходит к использованию JavaScript для создания интерактивных форм. Программа определяет подходящий товар на основе возраста пользователя и суммы денег, применяет систему скидок и отображает результат с изображением товара. Это задание демонстрирует работу с обработкой событий и условной логикой.

Четвёртое задание "dom-тест" представляет собой полноценную мини-викторину с вопросами о молодёжном сленге. Вопросы перемешиваются случайным образом, ответы проверяются автоматически, а после завершения доступен просмотр всех вопросов с правильными ответами и объяснениями. Это задание показывает работу с массивами и функциями.

Пятое задание "Разбор строки" демонстрирует работу с Drag & Drop API и обработкой строк. Пользователь вводит строку, которая разбирается на слова, числа и элементы с заглавными буквами. Элементы можно перетаскивать между блоками, менять их порядок и отслеживать историю взаимодействий. Это задание показывает работу с событиями перетаскивания, позиционированием элементов и управлением состоянием.

Все задания объединены единым минималистичным дизайном с современными градиентами, плавными анимациями и адаптивной вёрсткой. Сайт использует семантическую HTML5-разметку, CSS Grid и Flexbox для создания гибких макетов, а также демонстрирует правильную организацию файловой структуры проекта.
